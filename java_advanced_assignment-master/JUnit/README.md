# JUnit Testing Module

This module demonstrates comprehensive JUnit 5 testing techniques with practical examples.

## Features

### Calculator Testing
- Basic arithmetic operations (sum, subtract, multiply)
- Unit test cases with assertions
- Edge case testing

### Student Service Testing
- Student management operations
- Parameterized tests using CSV data source
- Testing with external data files

## Test Classes

- `CalculatorTest.java` - Tests for Calculator operations
- `StudentServiceTest.java` - Tests for Student service with parameterized inputs

## Running Tests

```bash
mvn test
```

## Dependencies

- JUnit 5 (Jupiter)
- Maven Surefire Plugin

## Test Coverage

All public methods are covered with unit tests including edge cases and expected exceptions.
